# TekVotes - Online Voting Platform

## 🎯 Overview

TekVotes is a modern online voting system for beauty pageants and contest events. It allows users to vote for their favorite contestants, submit nominations, and view live results.

## ✨ Features

- 👑 **Contestant Management** - Add, edit, delete contestants with images
- 🎪 **Event Management** - Create and manage events with categories
- 💰 **Secure Payments** - Paystack integration for vote purchases
- 📊 **Live Results** - Real-time voting results with admin toggle
- 📝 **Nomination System** - Users can nominate contestants
- 📱 **Mobile Responsive** - Works on all devices
- 🔐 **Admin Dashboard** - Full analytics and management
- 📧 **Email Notifications** - Get notified on nominations

## 🚀 Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Node.js, Express.js
- **Database**: MySQL
- **Payment**: Paystack API
- **Hosting**: Render.com
- **Email**: Nodemailer (Gmail SMTP)

## 📋 Prerequisites

- Node.js (v18+)
- MySQL (v8+)
- Paystack account (for payments)
- Gmail account (for email notifications)
